    Some of my projects

    Bolt Tunes

    A music catalog app built using HTML, CSS and JavaScript. It uses the MusicBrainz and YouTube APIs to retrieve results based on given input. The user can also search for an artist/band information and albums. I collaborated on this project doing all sorts of tasks.

    Mais Imobiliaria

    Real Estate agency app built using object-oriented PHP. It has features like: custom search, client registration and login and an admin area. My tasks on this project development were more front-end oriented.

    Paradise Guide

    This project built using PHP allows tourists to find and reserve any kind of activities in the São Miguel island. An admin can accept or decline such reserves. The main focus of the app is the security aspect of it: preventing Cross-site scripting, SQL injection, protocols awareness or data encryption and decryption.

    Super Sheep

    A game built using the Unity engine, where the hero is a sheep with electric powers that needs to defeat its enemies to reach the end of the level ("Super Mario" style). I was part of the dev team responsable for using the engine (and C#) to bring the characters to life.

    Bird Watching app

    Interactive table built using Ionic and Angular for the front-end (aspect that I worked on) and Django for the back-end. The main focus of it is the bird watching activity on the island of Corvo. Important locations, fauna and flora were themes also represented. The code is not avaliable due to privacy reasons, images on the demo also unavailable.

    Job Searching app

    Built using Ionic and Angular for the front-end (aspect that I worked on) and Django for the back-end, this interactive table refers to job offers and programs from some islands of the Azores archipelago. Due to privacy reasons, the API data (search fields and results) is not avaliable.
